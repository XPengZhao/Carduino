# 函数结构

1. 首先注意小车的控制包括三方面：
   - 角度的控制
   - 速度的控制
   - 转向的控制
2. 其次注意该函数中，控制语言的实现是通过系统的中断来完成,而并非在loop函数完成。
3. 主程序部分位于HJ-R3文件中，给出的库函分别作用如下：
   - BalanceCar   包含速度pid  转向pid 以及pwm电机输出 
   - DistanceSRF04   超声传感器代码
   - I2Cdev1     IIC通讯，扩展内容，可做了解
   - IRremote   底层中断代码
   - KalmanFilter   卡尔曼滤波，可做了解
   - MPU6050        mpu6050传感器程序，了解功能即可
   - MsTimer2       我们用的中断设置
